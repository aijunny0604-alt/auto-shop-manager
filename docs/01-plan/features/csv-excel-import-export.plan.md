# CSV/Excel 업로드 및 다운로드 기능 계획서

## 1. 개요

빅스모터스에서 사용 중인 구글 시트(CSV/Excel) 데이터를 재고관리 시스템에 직접 업로드하여
기존 데이터를 그대로 이관하고, 시스템에서 이어서 사용할 수 있도록 하는 기능.
또한 현재 시스템의 데이터를 Excel 파일로 백업/다운로드하는 기능을 포함.

## 2. 배경

- 빅스모터스는 현재 구글 시트로 예약관리, 매입매출(오일) 등의 데이터를 관리 중
- CSV 양식 구조:
  - 예약일자, 상태, 소유차량, 차량번호, 시작시간, 종료시간
  - 연락처, 금액, 수리내용, 비고1~3
  - 항목1~5 + 각 사용량, startDate, endDate
- 이 데이터를 auto-shop-manager 시스템으로 이관하여 통합 관리 필요

## 3. 목표

1. **CSV/Excel 파일 업로드**: 빅스모터스 구글시트 양식의 CSV/Excel 파일 업로드
2. **자동 파싱 & 매핑**: 업로드된 데이터를 시스템 데이터 모델에 자동 매핑
3. **데이터 미리보기**: 임포트 전 데이터 미리보기 및 매핑 확인
4. **DB 저장**: 파싱된 데이터를 SQLite(Prisma)에 저장
5. **이어서 사용**: 임포트 후 시스템에서 바로 이어서 사용 가능
6. **Excel 다운로드**: 현재 데이터를 Excel 파일로 백업 다운로드

## 4. 기능 범위

### 4.1 임포트 (CSV/Excel → 시스템)
- CSV (.csv) 및 Excel (.xlsx, .xls) 파일 업로드 지원
- 빅스모터스 양식 자동 인식 (헤더 행 감지)
- 컬럼 매핑 미리보기 화면
- 중복 데이터 감지 (차량번호 기준)
- 업로드 이력 관리

### 4.2 엑스포트 (시스템 → Excel)
- 재고 목록 Excel 다운로드
- 예약/서비스 이력 Excel 다운로드
- 빅스모터스 양식 호환 포맷 지원

## 5. 기술 스택

- **파일 파싱**: xlsx (SheetJS) 라이브러리 - CSV와 Excel 모두 지원
- **파일 다운로드**: file-saver 라이브러리
- **백엔드**: Next.js API Routes + Prisma
- **프론트엔드**: React + Tailwind CSS

## 6. 성공 기준

- [ ] 매핑된 데이터 미리보기 화면 정상 표시
- [ ] DB 저장 후 재고관리 페이지에서 데이터 확인 가능

| 리스크 | 대응 |
|--------|------|
| CSV 인코딩 문제 (한글 깨짐) | UTF-8, EUC-KR 등 다중 인코딩 지원 |
| 빅스모터스 양식 변경 시 | 유연한 헤더 매핑 로직으로 대응 |
| 대용량 파일 업로드 | 청크 단위 처리, 진행률 표시 |
